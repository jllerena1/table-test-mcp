# Second Commit - Scripts de Automatización GitHub

## ¿Qué hice?

Agregué scripts de automatización para optimizar el proceso de subida de proyectos a GitHub y facilitar las actualizaciones futuras.

### Scripts Implementados

1. **COMANDOS_GITHUB.sh**
   - Script completo para configuración inicial de repositorio
   - Verifica instalación de Git
   - Inicializa repositorio si es necesario
   - Configura remote y rama principal
   - Guía paso a paso para el usuario

2. **update.sh** (nuevo)
   - Script rápido para actualizaciones futuras
   - Verifica cambios antes de proceder
   - Muestra estado del repositorio
   - Automatiza: add, commit y push
   - Manejo de errores mejorado

### Funcionalidades de los Scripts

- ✅ Verificación automática de Git
- ✅ Detección de cambios pendientes
- ✅ Mensajes de commit interactivos
- ✅ Validación de configuración
- ✅ Manejo de errores
- ✅ Interfaz con colores para mejor UX
- ✅ Instrucciones claras paso a paso

### Mejoras en el Flujo de Trabajo

**Antes:**
- Ejecutar múltiples comandos manualmente
- Recordar la secuencia de comandos
- Configurar remote manualmente cada vez

**Ahora:**
- Un solo comando para configuración inicial
- Un solo comando para actualizaciones
- Proceso automatizado y guiado

### Tecnologías Utilizadas

- Bash scripting
- Git commands
- Terminal colors (ANSI)
