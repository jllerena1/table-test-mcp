# First Commit - Data Table Component

## ¿Qué hice?

Este proyecto es una tabla de datos creada desde un diseño de Figma con las siguientes características:

### Funcionalidades Implementadas

1. **Tabla de Datos Completa**
   - 8 columnas con encabezados
   - Filas seleccionables con checkboxes
   - Resaltado de filas seleccionadas

2. **Dark Mode**
   - Toggle para cambiar entre modo claro y oscuro
   - Transiciones suaves entre temas
   - Todos los componentes adaptados

3. **Barra de Acciones por Lotes**
   - Aparece cuando hay filas seleccionadas
   - Muestra el conteo de elementos seleccionados
   - Botones de acción disponibles

4. **Paginación**
   - Selector de items por página
   - Navegación entre páginas
   - Contador de items

### Tecnologías Utilizadas

- React 18
- Vite
- CSS puro (sin frameworks externos)

### Proceso de Desarrollo

1. Diseño obtenido desde Figma
2. Componente creado con React
3. Estilos implementados con CSS
4. Dark mode agregado
5. Proyecto subido a GitHub
